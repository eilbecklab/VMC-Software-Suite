Add VMC identifiers to your VCF file

All identifiers are generated using the VMC digest algorithm (see specification) and will be added to the INFO field of each variant in the file. Variants will be right-normalized using the VT algorithm, as required by the VMC specification. The following descriptors will also be added to the meta-information section of the file.
